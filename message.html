Messagerie¶

Cette section vous permet de consulter les messages liés à l’application. Selon les préférences utilisateur, les messages reçus peuvent générer des mails envoyés sur l’adresse mail du compte.

Les types de messages¶

Il existe 2 types de messages dans l’application :

  • les messages d’invitation
  • les messages de notification

Les messages d’invitation¶

Ces messages sont envoyés par un utilisateur à un autre afin de rejoindre une équipe. Lorsqu’un utilisateur reçoit une invitation, il doit traiter le message avant de pouvoir le supprimer. Une invitation peur être acceptée ou refusée.

Les messages de notification¶

Ces messages sont envoyés pour chaque entrée ajoutée sur un formulaire sur lequel vous avez décidé de recevoir des notifications. Le message propose un lien vers l’entrée ajoutée.

L’écran de consultation des messages¶

Cet écran, accessible depuis la barre de navigation, vous permet de gérer vos messages. Il liste tous les messages reçus, les plus récents en premier. Les informations affichées sont les suivantes :

  • le type de message
  • le sujet du message. Vous pouvez cliquer pour accéder aux détails
  • la date de réception du message
  • l’expéditeur du message

Il est possible de supprimer un message en cliquant sur la croix en fin de ligne. Cette croix n’est pas affichée si le message n’a pas été traité.
